# Thiranex Internship: Data Cleaning & Visualization

This project is part of the Thiranex Internship. It involves data cleaning, exploratory data analysis (EDA), and visualization of an e-commerce dataset.

## Contents

- **`Data_Cleaning.ipynb`**: A Jupyter Notebook containing the code for loading, cleaning, and visualizing the dataset.
- **`Predictive_Modeling.ipynb`**: A Jupyter Notebook containing the machine learning code to predict customer recommendation based on product reviews and other features.
- **`WomensClothingE-CommerceReviews.csv`**: The dataset used for this analysis, containing Women's E-Commerce Clothing Reviews.

## Setup Instructions

1. Ensure you have Python installed.
2. Install the required libraries (such as `pandas`, `matplotlib`, `seaborn`, and `jupyter`).
3. Open the `Data_Cleaning.ipynb` and `Predictive_Modeling.ipynb` notebooks using Jupyter to view the code and visual outputs.
